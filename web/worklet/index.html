plyphon (AudioWorklet)

The same examples as the default build, but driven by cpal's AudioWorklet backend - audio runs on a dedicated Web Audio worklet thread (off the deprecated ScriptProcessorNode) via WASM threads and SharedArrayBuffer.

Synthesis

  • sine - the simplest example: a continuous 440 Hz sine.

More examples are migrated to this backend during rollout. Run any example natively with cargo run -p example-<name>. See the repository for details.
