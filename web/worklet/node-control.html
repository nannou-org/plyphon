node-control

In-graph node-to-node control. A chord of sustained drones is driven by separate gater nodes that address each drone by id: Pause.kr(LFPulse.kr(rate), id) pauses and resumes it on every gate edge, so the voices pulse against each other at their own rates, and Free.kr ends it. The node notifications (/n_off, /n_on, /n_end) print to the console.

Click anywhere to start audio.
